# MMVB Signal

Небольшой фронтенд-проект: приветственный лендинг + форма анализа сигналов по акциям ММВБ.

## Что внутри

- `курсор2.html` - главная страница (лендинг + форма)
- `styles.css` - стили интерфейса
- `script.js` - логика расчета сигнала и режимы (1-клик, ТОП-3, ежедневный отчет)

## Быстрый старт

1. Откройте `курсор2.html` в браузере.
2. Заполните параметры анализа.
3. Нажмите `Сформировать результат`.

## Дополнительно в интерфейсе

- История последних расчетов сохраняется в `localStorage`.
- Чекбокс `Только сильные сигналы` скрывает все `умеренный/слабый` карточки.
- Кнопка `Экспорт отчета (.html)` скачивает файл отчета одним кликом (с учетом текущего фильтра).

## Логика

- Встроен анти-слив фильтр: при слабых условиях всегда `НЕТ СИГНАЛА`.
- Учитываются: уровень, объем, структура, свеча, тренд, фаза, RR, индикаторы.
- Сильный сигнал может дать `BUY` или `SELL`, иначе возвращается `НЕТ СИГНАЛА`.
